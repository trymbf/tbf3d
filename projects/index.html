BALANCING BROS

My own LAN multiplayer game to play with friends.

Balancing Bros is a LAN multiplayer game entirely written in Python. It uses pyMunk for Physics, PyGame for visuals and PyGameUI for the UI. You play as a ball trying to balance on a rotating platform. The last ball standing wins. The controls are really simple: roll right, roll left and jump. These easy controls make it easy to pick up and produce lots of fun!

Download (.exe)

GEOGEBRA DASH

GeoGebra has a surprising amount of functionality.

For some reason GeoGebra has a programming language built in, GeoGebraScript. When I realised this I saw an opportunity. It does not support keyboard inputs, but if you force focus on a slider, the right and left arrows become functional. GeoGebraScript also translates to the selected language which makes development a huge pain, as the GeoGebra app uses British English while the web version uses American. Still, the game is quite fun, functional and addicting.

Play GGB Dash!

PYGAMEUI

Create full PyGame GUIs in minutes.

PygameUI is a small Python library written in Python that makes it easier to create UI elements like buttons, texts, inputs, and more when using the Python library PyGame. The library also supplies the user with many optional features for customising the UI elements. If you are looking for a library that makes small projects and games a lot less of a headache, I would definitely recommend you give PyGameUI a try!
